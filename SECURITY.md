# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them through GitHub's [private vulnerability reporting](https://github.com/wingnut128/rustop/security/advisories/new) feature.

You should receive a response within 24 hours. If the issue is confirmed, we will:
1. Acknowledge the report
2. Work on a fix
3. Publish a security advisory

For other security-related questions, please open a GitHub issue.
